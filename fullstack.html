FULL-STACK WEB APPLICATIONS

Animeworld

Anime World is a full-stack web application built with Flask and PostgreSQL. It displays products dynamically from a database and allows users to place orders. Orders include shipping details and automatically calculate total cost based on region.
The system updates product stock after each order and stores all data in PostgreSQL. It also includes a contact form for sending messages stored in the database.
